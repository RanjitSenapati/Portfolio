# Personal Portfolio Website

A responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Sections
- About
- Skills
- Projects
- Contact

## How to Run
Just open `index.html` in your browser. No build tools needed.

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, animations)
- Vanilla JavaScript
